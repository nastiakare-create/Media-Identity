# План: деплой на Railway

## Цель
Подготовить проект к развёртыванию на Railway и дать чёткую инструкцию, как подключить репозиторий и включить автодеплой.

## Фазы

- [x] Оценить текущее состояние репозитория
- [x] Создать минимальный развёртываемый сервер и статическую страницу
- [x] Добавить конфигурацию Railway (`Procfile`, `package.json`, `.gitignore`)
- [ ] Инициализировать Git и подготовить репозиторий к пушу на GitHub
- [ ] Дать точную инструкцию для подключения к Railway и запуска автодеплоя

## Итог
Проект готов к подключению к Railway, но для финального развёртывания нужно:

1. создать репозиторий на GitHub
2. запушить текущие файлы
3. подключить GitHub к Railway
4. включить автодеплой
